    20170606
        レベル分けを行う。
        リズムゲーム
        好感度パラメータ管理
        発言によって雰囲気・体力パラメーター変動
        雰囲気が悪くなりすぎると追放、体力が0になると死亡
        モードは、女子会、メンヘラ受け流し、繰り返される友人Aの自慢話、上司の愚痴、就職面接
        ステージごとに曲を作成する。
        内心つけるか
        発言は最大6種類
        途中でピンチモードを入れて、単調さを無くしたい
       
    20170608
    
        モードを女子会で統一した方が良いかも（「女子会」のテーマ受けが良い、作画、セリフが一点集中できる、上司の愚痴とかよく分からない）
        発話のタイミングについて、会話文字列が画面外から飛んできて、中央で停止した瞬間からカウントを開始する
        会話では体力が減る一方で、カウント外時間に「内心」をつぶやくことによって回復する形式はどうだろうか
        会話はジャンル分けし、発話による雰囲気の上下はジャンルごとに統一させる。
        現在１２種類の発話を考案。（多分これ以外いらない）
        ステージや難易度をどのように変えるかは検討中。新キャラや初期条件や音楽の速さ？
        
    20170613
    
        ・女子会リズムゲームシステムは以下の通りである
        ・CPUの発話とBGMのリズムに合わせて黄色マークが出るので、それに従って相槌ボタンを選択してクリックする。
        ・正しいタイミングで正しい相槌を打つと雰囲気ゲージが上昇する一方、そうでない場合は雰囲気ゲージが下降する。
        ・相槌をすると体力ゲージが下降する。下降度は相槌の内容に依存する。
        ・発話とは無関係に青色マークが出るので、その時に「内心」をつぶやくことで体力を回復する。回復量は内心の内容に依存する。
        ・赤色マークが出た時、そのマークの終了後、相槌ボタンの位置が変化する。
        ・雰囲気ゲージが0になると「追放」され、体力ゲージが0になると「死亡」する。（どちらもゲームオーバー）
        ・ステージ（難易度）は4種類。それぞれクリアするごとに、そのステージのエンドレスモードが遊べるようになる。
        ・難易度が上昇するごとに、相槌ボタンの選択肢が増えていく
        ・エンドレスモードは一周する度に曲が早くなる。
